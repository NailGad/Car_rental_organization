# Структура проекта - Этап 1 (Фундамент)

## Что сделано:

### 1. База данных
- **SQLite** - выбрана как основная БД
- Файл БД создается автоматически в `AppDataLocation/car_rental.db`
- Singleton паттерн для DatabaseManager

### 2. Модели данных (4 класса)
- **User** - пользователь (клиент/администратор)
- **Car** - автомобиль (марка, модель, статус, цена)
- **Rental** - аренда (даты, стоимость, статус)
- **Fine** - штраф за просрочку

### 3. DatabaseManager
- Полный CRUD для всех сущностей
- Методы поиска (по марке, дате, клиенту)
- Автоматическое создание таблиц
- Создание администратора по умолчанию (admin/admin123)

### 4. Структура проекта
```
Organization/
├── models/          - классы сущностей
├── database/        - работа с БД
├── main.cpp         - точка входа с инициализацией БД
└── mainwindow.*     - главное окно (базовая структура)
```
